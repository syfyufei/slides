Covariate analysis

Where the coefficient come from

Probability: The chance of occurrence of Xi given the PDF of X, P(O|θ).

Likelihood : How probable a given set of observations is for certain values of the parameters of a distribution, ℒ(θ|O) = \(\prod_{i=1}^n y_i\)

ℒ Lagrangian or Laplace transform

Formally, let O be the set of observed outcomes and θ be the set of parameters that describe the stochastic process.

Probability: The integral, the area, the results.

Likelihood: The parameter, the product, the hypothesis.

ℒ can above 1.

Maximum Likelihood Estimation: Maximizing the probability that ℒ(θ|O) = P(O|θ)

  • Goal: Model (parameter) selection
    • What’s the most appropriate estimation?
      • Maximizing the probability that ℒ(θ|O) = P(O|θ)
    • Point estimation: Unbiasdness
    • Interval estimation: Range of plausible values
    • Goodness of fit: Explained variances
    • Diagnostic estimation: What if the assumptions are violated

E.g., Given binomial distribution \(L(\pi) = {n \choose s}\pi^s(1 - \pi)^{n - s}\), what π reaches the maximum likelihood?

\[log[L(\pi)] = log{n\choose s} + s\cdot log(\pi) + (n - s)\cdot log(1 - \pi).\]

To get the peak value, we use the derivative:

\[\frac{dlog[L(\pi)]}{d\pi} = \frac{s}{\pi} - \frac{n - s}{1 - \pi}.\]

To minimize this, let it equal 0

\[\begin{align} \frac{s}{\pi} - \frac{n - s}{1 - \pi} = 0 \Rightarrow \frac{s}{\pi} &= \frac{n - s}{1 - \pi},\\ s(1 - \pi) &= \pi(n - s),\\ s - s\pi &= n\pi - s\pi \Rightarrow\pi = \frac{s}{n}. \end{align}\]

Statistical Pillas

  • Law of Large Number (LLN)
  • Central limit theorem (CLT)

Two Pillas

LLN: For a sample with size n of a random variable X,

\[{\displaystyle \lim _{n\to \infty }\sum _{i=1}^{n}{\frac {X_{i}}{n}}= \mu.} \]

When n gets larger, the ratio of outcomes, X̄, will converge to the theoretical (population) average μ

Converge = concentrated around

Proof: https://blog.csdn.net/a19990412/article/details/85283729

CLT: For a random sample of n, X̄ fluctuate around μ with an uncertainty,

\[When\ n \rightarrow +\infty, Pr(\bar X) \sim \mathcal{N(\mu, \sigma)}. \]

When independent random variables are summed up, their properly normalized sum tends toward a normal distribution, even if the original variables themselves are not normally distributed.

Proof: https://zhuanlan.zhihu.com/p/93738110

When n gets larger…

‍CLT: X̄ will distribute normally;
‍LLN: X̄ will approach to μ

Contradictory?

Becoming a value or becoming a distribution?

CLT shows the the shape, LLN shows the location of the center.

Conduct Hypothesis testing

Procedure

  1. An observed sample;
  2. Making the IID assumption;
  3. Set up the null (and alternative) hypotheses
  4. Set the confidence level;
  5. Compare the X̄ with &mu.

IID

  1. An observed sample;
  2. Making the IID assumption;
    • Identical: X and Y are from the same distribution, \({\displaystyle F_{X}(x)=F_{Y}(x)\,\forall x\in I}\)
    • Independent: \({\displaystyle F_{X,Y}(x,y)=F_{X}(x)\cdot F_{Y}(y)\,\forall x,y\in I}\)
  3. Set up the null (and alternative) hypotheses
  4. Set the confidence level;
  5. Compare the X̄ with &mu.

Unbiased point estimate:

\[\begin{align} Given \bar X = \frac{\sum X}{n}, E(\bar X) =& \frac{1}{n}[E(X_1) + E(X_2) + \dots + E(X_n)],\\ =& \frac{1}{n}[\mu + \mu + \dots + \mu] = \mu. \end{align}\]

Unbiased variance (standard error):

\[\begin{align} s(\bar X)^2 =& \frac{[E(s(X_1)^2) + E(s(X_2)^2) + \dots + E(s(X_n)^2)]}{n^2}\\ =& \frac{1}{n}[\sigma^2 + \sigma^2 + \dots + \sigma^2] = \frac{\sigma^2}{n}\\ \therefore\ SE =& \frac{\sigma}{\sqrt{n}}. \end{align}\]

Hypothesis Setting

  1. An observed sample;
  2. Making the IID assumption;
    • Identical: X and Y are from the same distribution, \({\displaystyle F_{X}(x)=F_{Y}(x)\,\forall x\in I}\)
    • Independent: \({\displaystyle F_{X,Y}(x,y)=F_{X}(x)\cdot F_{Y}(y)\,\forall x,y\in I}\)
  3. Set up the null (and alternative) hypotheses
    • H0: Specifying values for one or more population parameters in a random distribution (μ, π rather than X̄, P);
    • H1: the population parameter is something other than the value in the stochastic status;
  4. Set the confidence level;
  5. Compare the X̄ with &mu.

Confidence Interval

  1. An observed sample;
  2. Making the IID assumption;
    • Identical: X and Y are from the same distribution, \({\displaystyle F_{X}(x)=F_{Y}(x)\,\forall x\in I}\)
    • Independent: \({\displaystyle F_{X,Y}(x,y)=F_{X}(x)\cdot F_{Y}(y)\,\forall x,y\in I}\)
  3. Set up the null (and alternative) hypotheses
    • H0: Specifying values for one or more population parameters in a random distribution (μ, π rather than X̄, P);
    • H1: the population parameter is something other than the value in the stochastic status;
  4. Set the confidence level;
    • 1 - α
  5. Compare the X̄ with &mu.

Standard Error (Formal Introduction)

\[SE(\bar X) = \frac{s}{\sqrt{n}}.\]

E.g. Given the population mean as 69 and standard deviation as 3.2, how would the mean of a random sample of four observations fluctuate?

\[\begin{align} E(\bar X) =& \mu = 69; \sigma = 3.2, \\ \therefore SE(\bar X) =& \frac{3.2}{\sqrt 4} = 1.6. \end{align}\]

SE and Z-Score

  • Individual-level Data
  • Aggregate Data

\[Z = \frac{\bar X - \mu}{SE}= \frac{\bar X - \mu}{\sigma/\sqrt n}.\]

E.g. Given \(\mu\) = 72 and \(\sigma\) = 9, and a random sample of 10. Calculate the probabilities of P(X > 80) and P(X̄ > 80)

\(Z = \frac{80 - 72}{9} = .89\Rightarrow P(Z > .89) =\) 0.1867;

\(Z = \frac{80 - 72}{9/\sqrt{10}} = 2.81\Rightarrow P(Z > 2.81) =\) 0.0025.

\[Z = \frac{P - \pi}{\sqrt{\frac{\pi(1 - \pi)}{n}}}.\]

In a sample of N, the sample proportion P fluctuates around the population proportion π with SE, \(\sqrt{\frac{\pi(1 - \pi)}{n}}\). (Why?)

Proof: \(\sigma^2 = E(X^2) - \mu^2 = \pi - \pi^2 = \pi(1 - \pi)\). ∴ \(\sigma = \sqrt{\pi(1 - \pi)}, SE = \sigma/\sqrt{n} = \sqrt{\frac{\pi(1 - \pi)}{n}}.\)

Then, \(Z = \frac{P - \pi}{\sqrt{\frac{\pi(1 - \pi)}{n}}}\), the confidence interval: \(\pi = P \pm Z\sqrt{\frac{P(1 - P)}{n}}.\)

E.g., Given the Republican are 60% of the population, while the Democrats are 40%. What’s the probability that the Republican are the minority in a random sample of 100 people from the population?

Minority means \(P(\pi < 0.5)\).

Then, \(Z = \frac{0.5 - 0.6}{\sqrt{\frac{0.6(1 - 0.6)}{100}}} =\) 2.0412, therefore, P(Z < 2.0412) = 0.0206.

Correcting Small Population

Remember sampling w.o. replacement?

Without replacement, SE of X̄ is reduced by \(\sqrt{\frac{N - n}{N - 1}}\) (finite population correction, FPC) ⇒ More uncertainty.

Illustration:

  • n = 1 (sampling only one sample), Then \(SE = \frac{\sigma}{\sqrt{n}}\sqrt{\frac{N - n}{N - 1}} = \frac{\sigma}{\sqrt{n}}\), no changes;
  • n = N (sampling all), SE = 0.
  • n = 1000, N = 100,000,000 (large sample of a large population), \(FPC = \sqrt{\frac{100000000 - 1000}{100000000 - 1}}\approx .999\), little changes.
  • n = 100, N = 108 (large sample of a small population), \(FPC = \sqrt{\frac{108 - 100}{108 - 1}}\approx 0.075\), some changes.

Example

Given a virus can influence 10% of the population. Now there’s a sample of senior people, n = 527, within which there are 14% infected the virus. Are senior people more likely to be victimized?

  • Assuming the sample is IID sampled and \(H_0: \pi \leq 10; H_1: \pi > 10.\)
  • α = 0.05
  • Confidence-interval method:
    • \(\pi = 0.14 \pm 1.96 * \sqrt{\frac{0.14 * (1 - 0.14)}{527}} = 0.14 \pm 0.03,\) that is [0.11, 0.17] > 0.1. \(H_0.\) rejected.
  • Critical value method
    • \(Z_{obs} = \frac{P - \pi}{\sqrt{\frac{\pi(1 - \pi)}{n}}} = \frac{14 - 10}{\sqrt{\frac{0.1 * 0.9}{527}}} = 3.06.\) Given the level of $= 0.05, Z_{critical} = $ 1.96 < \(Z_{obs}\),1 therefore reject the \(H_0.\)
